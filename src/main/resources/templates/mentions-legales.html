2. Responsable de publication

Le responsable de publication est le représentant légal de l’association Les Magiciens d’Abord.

L’application est hébergée sur un environnement technique interne sécurisé, administré pour les besoins de l’association.

4. Finalité de l’application

MagicLibrary est un outil interne permettant :

  • la gestion des objets de la bibliothèque réelle.
  • la gestion des emprunts.
  • l’administration des comptes utilisateurs.
  • la communication interne via le formulaire de contact.

L’application n’a aucune vocation commerciale.
